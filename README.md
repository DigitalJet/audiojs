# Audiojs

[audio.js](http://kolber.github.com/audiojs/) is a drop-in javascript library that allows HTML5's <audio> tag to be used anywhere.

## Installation

Add this line to your application's Gemfile:

    gem 'audiojs'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install audiojs
